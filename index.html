HoloCognitive

Determine a person's emotion with HoloLens and Azure Emotion API


This application determines the emotion of people around using Azure Emotion API. A person's face is scanned using HoloLenses camera and analyzed in the cloud. Output is shown via emoji overlay directly on top of the person's face.

  • Date: January 2018
  • Category: HoloLens, Azure

HoloChatting

Chatting application with HoloLens


This is a chatting application for HoloLens and PC. Communication is done using websocket. On the HoloLens side, user can directly talk to the microphone and the application can directly recognize and translate them to text. Supports Japanese language.

  • Date: November 2018
  • Category: HoloLens
